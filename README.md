## skyChat

一个使用 Flask 构建的极简，私密web聊天应用程序！

用到的技术栈大部分都是flask，现支持如下:

- 多用户实时聊天



### 使用

```sh
pip3 install flask_socketio flask eventlet
nohup python app.py &
```

open `http://127.0.0.1:8888` to play with skyChat.
